== Description ==

A sidebar widget that adds a beautiful watch to your wordpress blog which shows your visitors local time.

**Features**<br />
-includes built-in widget<br />
-add your own header and bottom text
-enables a beautiful watch

== Installation ==

1. Download the plugin
2. Unzip watches-capital-watch.zip
3. Upload the entire watches-capital-watch folder to your plugins directory
4. Go to the Plugins page in your WordPress Administration area and click to 'Activate' "Watches Capital Watch" on WordPress.
5. Go to Appearance->widgets and add the "Watches Capital Watch" sidebar widget on WordPress widget to your sidebar

Please let me know any bugs, improvements, comments, suggestions.
